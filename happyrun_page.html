problem

작은 텍스트와 한 곳으로 몰려있는 버튼.

직관적이지 않고 자세히 봐야 하는 UI·UX

불편한 인터페이스 등 사용자들이 불편함을 느낄 소지가 다수 발견되어 선정

swot 분석

strength

간결한 기능.
필수 요소만 있는 쉬운 구성

weakness

비효율적인 레이아웃으로 사용의 불편함.
가독성이 떨어지는 색상

opportunity

초보자들에게 운동을 가볍게 입문 시킬 수 있음.
마스크 착용 의무 해제로 운동 환경이 이전보다 조성.

threat

기존에 사용자들 사이에서 이미 유명하고
전문성을 갖춘 나이키, 런데이 등 어플이 많이 있음.

AS IS

작은 텍스트와 아이콘

텍스트와 아이콘이 작아서 사용자가
기능을 쉽게 확인하기 어려움

직관적이지 않은 레이아웃

자주 사용되는 버튼이 위쪽으로 배치되고
주요 기능이 한눈에 들어오지 않음

정보 전달에 적합하지 않은 색 배치

배경색과 글자색이 비슷하여 정보 전달에
적합하지 않으며, 보기 불편함
디자인 적인 색상이 없음

기록체크의 불편함

운동의 기록과 나의 상태 체크가
텍스트 한 줄로만 구성되어 내용 확인이 어려움

TO BE

텍스트 아이콘 크기 조절

텍스트와 아이콘의 크기를 조절하여
사용자가 쉽게 확인할 수 있게 함

레이아웃 수정

운동 중에도 사용이 쉽도록 직관적인 배치와
화면 구성 주요 기능과 부가적 기능의 분리

적절한 색 배치와 포인트 컬러 추가

전체적인 배색을 고려하여 가독성이 좋은 색
배치와 디자인적인 포인트 컬러 추가

기록 화면의 재구성

나의 기록 변화를 쉽게 비교할 수 있는
배치와 편의성을 주는 시각적인 요소의 사용
